# Atlanexis CloudOS - Ghayma Website

Standalone dark-mode marketing site for the project.

## Files

- `index.html`: main product landing page
- `enterprise.html`: enterprise support offer page
- `styles.css`: visual system, layout, and animation styles
- `main.js`: scroll reveal, ambient motion, and copy-to-clipboard behavior
- `assets/`: brand and illustration assets shared by both pages

## Open locally

You can open `index.html` directly in a browser, or serve the folder with any static file server.
